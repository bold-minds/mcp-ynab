# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, verbatim as our community code of conduct.

The full text is available at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- [`CODE_OF_CONDUCT.md` on the Contributor Covenant repository](https://github.com/EthicalSource/contributor_covenant/blob/release/content/version/2/1/code_of_conduct.md)

## Scope

The Covenant applies within all community spaces associated with this project — the GitHub repository, issue tracker, pull requests, discussions, and any other channels where project participants interact in connection with the project.

## Enforcement

Reports of conduct issues may be sent privately to the maintainers via GitHub's private contact mechanisms (e.g. a direct message to the repository owner, or — for conduct concerns that also involve a potential security matter — the channel described in [`SECURITY.md`](SECURITY.md)).

All reports will be reviewed and investigated promptly and fairly. The maintainers will respect the reporter's privacy and will not share details without consent, except as required to investigate and respond.

## Enforcement guidelines

The project uses the [Contributor Covenant Enforcement Ladder](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) to determine consequences for any action deemed in violation.
